## What is GraphQL?

```graphql
type Actor = {
  id: ID!
  name: String!
}

type Movie = {
  id: ID!
  name: String!
  actors: [Actor]
}

type Query = {
  movie(id: ID!): Movie
}
```

</L.CodeSlide>
